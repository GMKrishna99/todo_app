# Todo App

## Steps To Follow

**This App is Created in Vite & React**

*To install Vite & React*

```
npm install vite@latest
```
*Enter Project name*
```
project name:Todo-App
```
*go to Todo App*
```
cd Todo-App
```
*Install npm in this directory*
```
npm install
```
*to run project live*
```
npm run dev
```
